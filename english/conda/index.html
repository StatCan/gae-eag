    Anaconda & Python

    Anacoda

    Warning

    At this time, Anaconda Navigator is only offered in English.

    Anaconda is an open-source distribution of the Python and R that simplifies package management and deployment.

    Anaconda leverages the package management system, conda, which analyzes the current environment before executing and installation to avoid disrupting other frameworks and packages.

    Anaconda Navigator and allows users to launch applications and manage conda packages, environments and channels without using command-line commands.

    To create a new Environment using CMD window or python

    Create new environment (replace <__>): 

    conda create --name <env_name> <package>

    Activate your environment (replace <__>): 

    conda activate <env_name>

    Installing package(s): DAaaS utilizes Artifactory for package and library management: 

    https://jfrog.aaw.cloud.statcan.ca/artifactory/conda-forge-remote

    To use: 

    conda install -c https://jfrog.aaw.cloud.statcan.ca/artifactory/conda-forge-remote/ [package]

    For specific package version: 

    conda install -c https://jfrog.aaw.cloud.statcan.ca/artifactory/conda-forge-remote/ [package=X.X...]

    Example: 

    conda create --name geo
conda activate geo
conda install -c https://jfrog.aaw.cloud.statcan.ca/artifactory/conda-forge-remote/ geopandas
conda install -c https://jfrog.aaw.cloud.statcan.ca/artifactory/conda-forge-remote/ matplotlib=3.5.2
