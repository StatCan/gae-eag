    Comment soumettre une demande à ADS-Géo

    Pour obtenir du soutien technique ou présenter des demandes de modification, veuillez soumettre un billet JIRA:

    1. Ouvrez le logiciel JIRA de Statistique Canada: Plateforme géospatiale ADS (DASGEO)
    2. Dans la barre de menu, cliquez sur le bouton « Create » (créer):
    3. Dans la fenêtre Créer un ticket JIRA, sélectionnez le type de demande = Client Support:
    4. Dans la fenêtre Créer un ticket JIRA, saisissez les renseignements relatifs à votre demande :
      • Résumé [obligatoire] : Entrez un résumé de votre demande.
      • Responsable [obligatoire] : Sélectionnez automatique.
      • Composants [obligatoire] : Sélectionnez « Customer Support – Geo Platform User Help Resources :
      • Description [obligatoire] : Saisissez une description détaillée de votre demande.
      • Pièce jointe [facultatif] : Annexez des documents justificatifs relatifs à votre demande (p. ex. captures d’écran des erreurs).

    Le Service de soutien ADS-Géo remplira le reste des renseignements dans le billet JIRA.

    En priorité, un membre du Service de soutien ADS-Géo communiquera avec le client pour l’aider avec ses demandes.
